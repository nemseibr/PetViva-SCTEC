# 🐾 PROJETO PetViva LP

Projeto de uma landing page interativa para um Pet Shop, com catálogo de produtos, sistema de carrinho, busca, filtro e formulário de contato.
---

**Otávio Augusto Reis Nascimento**

---

## 🚀 Tecnologias Utilizadas

* HTML5 → Estrutura da aplicação
* CSS3 → Estilização e responsividade
* JavaScript → Lógica e interatividade

---

## 📌 Funcionalidades

* Carrinho de compras com persistência (localStorage)
* Busca de produtos por nome, categoria ou tipo
* Filtro por categorias (cachorro, gato, etc.)
* Catálogo dinâmico de produtos
* Modal informativo de serviços
* Formulário com validação em tempo real
* Layout responsivo (mobile e desktop)

---

## 🧠 Etapas de Desenvolvimento

### 1. Estrutura (HTML)

**Técnico:**
Construção da base da aplicação utilizando HTML semântico (`header`, `nav`, `main`, `section`, `footer`), organizando o conteúdo e permitindo integração com CSS e JavaScript.

**Simples:**
Montamos o “esqueleto” do site, como se fosse uma casinha de peças.

---

### 2. Estilização (CSS)

**Técnico:**
Aplicação de estilos com uso de variáveis globais (`:root`), Flexbox, Grid Layout e Media Queries para responsividade.

**Simples:**
Deixamos tudo bonito 🎨 e funcionando em celular e computador.

---

### 3. Lógica (JavaScript)

**Técnico:**
Manipulação do DOM, eventos, controle de estado do carrinho, renderização dinâmica de produtos, validação de formulário e uso de `localStorage`.

**Simples:**
Fizemos o site “ganhar vida” — agora ele responde quando você clica.

---

### 4. Carrinho de Compras

**Técnico:**
Sistema baseado em array com persistência via `localStorage`, permitindo adicionar, remover, atualizar itens e calcular o total.

**Simples:**
É uma sacolinha onde você guarda os produtos.

---

### 5. Busca e Filtros

**Técnico:**
Filtragem de produtos com `.filter()` e busca textual com `.includes()`.

**Simples:**
Você digita o nome do produto e o site encontra 🔍

---

### 6. Formulário

**Técnico:**
Validação com eventos (`input`), uso de regex para email e controle de estados visuais.

**Simples:**
Confere se você escreveu tudo certinho antes de enviar
